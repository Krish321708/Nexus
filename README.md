# 🌐 Nexus: Autonomous Cognitive Engine

**Nexus** is an API-driven intelligence architecture designed for maximum logical consistency, strategic output, and operational velocity. It acts as a force multiplier for your digital life, handling complex reasoning and automation via a streamlined, persistent interface.

---

## 🚀 Features

* **Modular Intelligence:** Decoupled architecture allowing for seamless model swapping.
* **Always-On Architecture:** Deployed as a persistent cloud service, ensuring 24/7 availability.
* **Telegram Integration:** A responsive, natural-language interface for instant strategic input.
* **Efficiency-First:** Engineered to cut through noise and deliver high-leverage outcomes.
* **Extensible:** Designed to evolve into a full-scale application or web-based UI.

## 🛠 Tech Stack

* **Core:** Python
* **Intelligence:** Novita AI (Llama 3.1 70B)
* **Infrastructure:** Render (Cloud Deployment)
* **Messaging:** Telegram Bot API
* **Version Control:** Git & GitHub

## 📦 Quick Start

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/](https://github.com/)[YOUR_GITHUB_USERNAME]/nexus.git
    cd nexus
    ```

2.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Configure Environment Variables:**
    Create a `.env` file in the root directory:
    ```env
    NOVITA_API_KEY=your_key_here
    TELEGRAM_BOT_TOKEN=your_token_here
    ```

4.  **Run:**
    ```bash
    python jarvis.py
    ```
